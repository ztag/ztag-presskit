### Product

Public product information for journalists and partners.

Contents:
- `overview.md`: What ZTAG is and who it serves.
- `features.md`: Key features and benefits.
- `photos/`: Approved imagery and diagrams (placeholders). Include alt text and captions.

Guidelines:
- Prefer SVG or high-resolution PNG for diagrams; JPG for photos.
- Name assets descriptively (e.g., `ztag-k12-checkin-station.jpg`).
- Include sources or attribution if required.
