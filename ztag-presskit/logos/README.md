### Logos and Brand Assets

This folder contains official ZTAG logo files and a placeholder brand guide.

Available files:
- `ztag-logo.svg` (vector)
- `ztag-logo.png` (raster, transparent background)
- `brand-guide.pdf` (placeholder)

Usage guidance:
- Maintain clear space equal to the height of the "Z" around the logo.
- Do not alter colors, proportions, or add effects.
- Prefer SVG for print and high-DPI; PNG for general web use.
