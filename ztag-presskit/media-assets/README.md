### Media Assets

Approved assets for press and partners. Organize by year/event. Do not add private or embargoed materials.

Structure:
- `YYYY/event-name/photos/` — high-resolution JPG/PNG
- `YYYY/event-name/pdfs/` — one-pagers, slide PDFs

Notes:
- Include descriptive filenames and, if needed, a companion `.md` with captions/credits.
- Keep originals unwatermarked; provide captions in alt text.
