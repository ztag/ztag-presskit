### ZTAG Press Kit

This public repository hosts ZTAG's official press kit. It provides journalists, analysts, partners, and event organizers with company background, product information, approved logos, media assets, and contact details.

#### How to use this press kit
- Browse folders for the assets and docs you need.
- Each folder has a README with what belongs there and how to use it.
- Cite public sources where indicated. Do not add private or embargoed materials.

#### Repository structure
- `about/`: Company facts, mission, founders.
- `logos/`: Official logos and brand guidelines.
- `product/`: Overview, features, and product photos/diagrams.
- `press-releases/`: Official releases and templates.
- `media-mentions/`: Articles, interviews, and coverage.
- `testimonials/`: Quotes and references.
- `community-programs/`: Community and education initiatives.
- `implementation-guides/`: Adoption guides and references (e.g., Bakersfield public docs).
- `conference-participation/`: Event outlines (e.g., BOOST) and talk materials.
- `media-assets/`: Event photos and PDFs organized by year/event.
- `contact/`: Press and partnership contact information.
